## **电压 = 电势差 = 电场的能量差**

- 电荷分布不均匀 → 产生电场
- 电场力推动电子从高电势运动到低电势
- 电荷分布 → 电场 → 电势是电场中某点的能量属性 → 电势差 → 电压

未接通电路时：

- 电场是电源电场

接通电路后：

- 导线和电阻中的电荷重新分布
- 整个电路产生一个新的电场

---

# **1. 电势（φ）是电场中某一点的能量属性**

**势能 = 一个力场在空间中制造的“能量地形”：**

- 高处 = 势能高
- 低处 = 势能低
- 在场的作用下：物体自然往低势能滑

![](https://cdn.nlark.com/yuque/__latex/e21e409d8abcba3c127f253a40ce4e46.svg)

它衡量：

“把单位正电荷从参考点移到该点，需要做多少功（或释放多少功）”。

补充性质：

- **势能是“反力场系统储存的能量”**
- 距离一定时，**场力越大 → 势能变化越大**
- 电势是标量场，只有大小，没有方向
- 在任何力场中：沿着力的方向移动，势能一定减少
- 势能减少量 = 力做的正功
- 力越大，势能减少越快

---

# **2. 电场（E）是电势的空间变化率（负梯度）**

- 电场由**电荷分布不均匀**产生
- 电场大小表示“推力大小”
- 电场力把电荷从高电势推向低电势
- 电场越强 → 推力越大 → W = Fd 做功越大 → 电势变化越快 → 电压越大

数学表达式：

![](https://cdn.nlark.com/yuque/__latex/fc47f053810c38909bc605c4d4b89937.svg)

含义：

- 电场方向 = 电势降低最快的方向
- 电场大小 = 电势下降的陡峭程度

比喻：

- 电势 = “高度”
- 电场 = “坡度”

---

# **3. 电场为 0 时，电势保持不变（为常数）**

- 同样两点之间：电场越强 → 电势下降越快 → 电压越大
- 电势是“高度”
- 电场是“坡度”

- 坡度越大 → 高度下降得越快 → 电势差（电压）越大
